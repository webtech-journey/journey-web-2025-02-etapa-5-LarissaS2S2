Habilidades Técnicas

CSS

O CSS serve para dar estilo essa estrutura, ajustando cores, fontes e layout, tornando o visual do site mais atrativo.

Bootstrap

Facilita criar interfaces consistentes e responsivas, tornando o desenvolvimento mais eficiente e adaptáveis a diferentes dispositivos.

GitHub

Uso o GitHub para versionar projetos, colaborar com equipes e compartilhar meu código de forma organizada e segura.
